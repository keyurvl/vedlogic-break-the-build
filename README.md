# Vedlogic QA Automation Engineering Challenge - Starter Project

This repository is an intentionally imperfect starter framework for a Selenium + Java + Cucumber + Maven + ReqRes challenge.

## Objective
Candidates are expected to:
- understand the existing codebase
- debug intentionally introduced bugs
- complete and improve UI and API automation
- keep code reusable and maintainable

## Applications Under Test
- UI: https://www.saucedemo.com/
- API: https://reqres.in/

## Tech Stack
- Java 8+
- Maven
- Selenium WebDriver
- Cucumber
- TestNG
- Rest Assured

## How to Run
```bash
mvn clean test
```

## Notes
- The framework contains intentional bugs and weak design choices.
- Page Objects are expected to be used and improved.
- Please document fixes and assumptions in your submission.
